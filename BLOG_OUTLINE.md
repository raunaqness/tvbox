# Build Your Own Personal Netflix with tvbox

## 1. What it is
A brief tour of the search-to-stream workflow and the finished setup.

## 2. What You Need
A VPS, domain and Cloudflare account, Google Drive storage, Infuse, Docker, and the required API keys.

## 3. Prepare the VPS
Secure the server, install Docker, clone tvbox, and configure environment variables.

## 4. Connect the Services
Configure TMDB, Prowlarr, aria2c, rclone, and the Google Drive destination.

## 5. Deploy tvbox
Start the Docker Compose stack and verify the web app, search, download, upload, and cleanup flow.

## 6. Add Secure Remote Access
Create a Cloudflare Tunnel, attach a hostname, and protect the app for browser access.

## 7. Set Up Infuse
Connect Infuse to Google Drive, select the media folder, and refresh the library.

## 9. Operate It Responsibly
Backups, updates, credential hygiene, VPS limits, and legal use.
